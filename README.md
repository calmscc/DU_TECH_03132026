NOLAlytics – AI Retail Visibility Analyzer
Description

NOLAlytics is an AI-powered tool that analyzes how often a retailer appears in AI-generated product recommendations. It combines a modern web interface with a backend AI system to:

Run visibility audits for a store-product combination.

Compare results across multiple AI engines.

Display results in interactive cards and charts.

Track previous audits in a history section.

This project integrates a front-end UI with a back-end AI engine to deliver a comprehensive analytics experience.

Features

Responsive sidebar navigation and model selection.

Real-time AI audit of store and product queries.

Summary cards: Prompts Tested, Best Engine, Highest Score.

Interactive visibility chart.

Audit history tracking for past queries.


How to Use

In the search box, enter a retailer and product (e.g., Walmart headphones).

Press Enter or click Analyze Visibility.

View the results:

Cards: Prompts Tested, Best Engine, Highest Score.

Chart: AI visibility percentages.

History: Previous audits.
